## Qué es GammaSync
Módulo `mod_SINCRO` (monorepo PROYECTO_MN, branch `OPTIMIZACION_TIEMPOS`, Python 3.13 + PyQt6 + numpy/scipy/pydicom/matplotlib/reportlab).
